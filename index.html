Completed projects

TOCHKA

TOCHKA is a very simple video game written in Haxe using the HaxePunk library. It was developed as part of my bachelor's thesis on cross-platform computer game development. The goal of the game is to dodge incoming obstacles for as long as possible in order to attain the high score.

The game's source code is freely available.
